### Chess Engine Built in Python
A chess engine made entirely from scratch in Python.

##  Feature List:
- [x] Basic board creation.
- [x] Piece Movement -
    - [x] Pawn
    - [x] Rook
    - [x] Bishop
    - [x] Knight
    - [x] Queen
    - [x] King
- [] Graphics with Pygame or p5.js
- [] State analysis -
    - [] Check for checkmate.
    - [] Check for stalemate.
